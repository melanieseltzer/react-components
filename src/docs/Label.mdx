# Label

An HTML label component. Should be used with [Input](/input).

## Props

<Props of={Label} />

## Basic usage

Label above input:

<Playground>
  <Flex direction="column">
    <Flex.Item>
      <Label className="custom-class" htmlFor="name2">
        Name
      </Label>
    </Flex.Item>
    <Flex.Item>
      <Input
        className="custom-class"
        id="name2"
        name="name2"
        placeholder="John Smith"
        border={1}
        width="100%"
      />
    </Flex.Item>
  </Flex>
</Playground>
