# @mels/react-components

A collection of some of my frequently used React components.

## Install

```bash
npm install --save @mels/react-components
```

## Use

```js
import React from 'react';
import { Card } from '@mels/react-components';

export const App = () => <Card>Hi!</Card>;
```

## Contribute

Run `yarn docz:dev` to start the Docz development server.
