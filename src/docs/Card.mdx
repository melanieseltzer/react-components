# Card

A super versatile Card component which is based on a Flex Item. Pass it any valid prop from Flex Item, in addition to a couple Card-specific props.

## Props

<Props of={Card} />

## Usage

### Default box-shadow

<Playground>
  <Flex>
    <Card flex="1 1 30%" background="tomato">
      Hi!
    </Card>
    <Card flex="1 1 30%" background="#fff">
      Hi!
    </Card>
    <Card flex="1 1 30%" background="lightsalmon">
      Hi!
    </Card>
  </Flex>
</Playground>
